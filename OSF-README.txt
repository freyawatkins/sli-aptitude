DATA:

`apt-data.csv` is the raw data file

`apt-data.rds` is the R data object resulting from the `00-wrangling-setup` script.


SCRIPTS:

`00-wrangling-setup` script contains a summary of the variables and missingness as well as pre-processing of the data file 

`01-group-analysis` script contains the selection and continuation analyses

`02-change-over-time` script contains the change over time plots and analyses for the predictor variables

`03-best-predictors` script contains the correlational plots and analyses between the predictor and outcome variables


MATERIALS:

Assessments used:

Dual N-Back (Jaeggi et al., 2008; from PEBL 2.0; Mueller, 2014; Mueller and Piper, 2014, http://pebl.sourceforge.net)

Corsi Blocks (Corsi, 1972; Kessels et al., 2000; from PEBL 2.0; Mueller, 2014; Mueller and Piper, 2014, http://pebl.sourceforge.net)

2D Mental Rotation (Shepard and Metzler, 1988; Berteau-Pavy, Raber and Piper, 2011; from PEBL 2.0; Mueller, 2014; Mueller and Piper, 2014, http://pebl.sourceforge.net)

Digit Span (Croschere et al., 2012; PEBL 2.0; Mueller, 2014; Mueller and Piper, 2014, http://pebl.sourceforge.net)

3D Mental Rotation (stimuli: Ganis & Kievit 2015; experiment task: https://pavlovia.org/freyawatkins/blockrotation)

Kirklees version of the Vernon Warden Reading Test (Hedderly, 1996; Warden, 1956). 

KBIT-2 Matrices sub-test (KBIT-2; Kaufman and Kaufman, 2004)

Barratt Impulsivity Scale (BIS-11; Patton, Stanford and Barratt, 1995)

MLAT Number Learning subtest(Carroll and Sapon, 1959; Carroll et al., 2010) 

British Sign Language Sentence Reproduction Test (Cormier et al., 2012)
